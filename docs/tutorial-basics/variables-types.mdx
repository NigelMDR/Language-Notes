## Cpp

Use ```./main > test.txt 2>&1 ``` to capture both stdout and stderr.

| Format | Description |
|:-------------------------------|:----------------------|
| ``` <TYPE> <Name> = <Value> ``` | initialize variables |
| ``` <TYPE> <Name>, <Name>, ...```| Declare variables |
| ``` <Name> = <Value> ```       | initialize declared variables |
